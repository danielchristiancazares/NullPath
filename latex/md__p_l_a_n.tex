Purpose
\begin{DoxyItemize}
\item Make Null\+Path a scientifically rigorous, reproducible null‑geodesic tracer and renderer that matches or exceeds current gold standards (e.\+g., Geo\+Kerr for geodesics; Gyoto/\+RAPTOR class for ray tracing) on accuracy, with clearly quantified error bars and validation artifacts.
\end{DoxyItemize}

Scope
\begin{DoxyItemize}
\item Primary focus\+: scientific accuracy ("{}research"{} mode). Non‑research/fast modes are out of scope for this plan.
\item Spacetimes\+: Schwarzschild (now), Kerr (next). Only vacuum metrics; matter effects and polarization are out of scope for this phase.
\end{DoxyItemize}

Outcomes
\begin{DoxyItemize}
\item A verified geodesic solver with tight, documented accuracy targets; reproducible comparison datasets vs. literature and established codes; images and tables that can be cited.
\end{DoxyItemize}

\hypertarget{md__p_l_a_n_autotoc_md23}{}\doxysection{\texorpdfstring{1) Definitions, Units, and Conventions}{1) Definitions, Units, and Conventions}}\label{md__p_l_a_n_autotoc_md23}

\begin{DoxyItemize}
\item Units
\begin{DoxyItemize}
\item Geometric units (G = c = 1). Report both M and Rs=2M units in logs and artifacts.
\end{DoxyItemize}
\item Impact parameter
\begin{DoxyItemize}
\item b ≡ L/E at infinity. Always additionally report b/M.
\end{DoxyItemize}
\item Critical scales (Schwarzschild)
\begin{DoxyItemize}
\item Photon sphere\+: r\+\_\+ph = 3M = 1.\+5 Rs.
\item Critical impact\+: b\+\_\+crit = 3√3 M = (3√3/2) Rs.
\end{DoxyItemize}
\item r\+\_\+min
\begin{DoxyItemize}
\item Turning radius (if it exists)\+: the largest positive real root of the radial turning-\/point equation.
\item For captured rays (no turning point), r\+\_\+turning is undefined; report NaN and a separate closest\+\_\+radius\+\_\+reached for diagnostics.
\end{DoxyItemize}
\item Deflection and lensing angles
\begin{DoxyItemize}
\item Store total Δφ along the path; define lensing bend α = Δφ − π for escaped rays.
\end{DoxyItemize}
\item Observer tetrads
\begin{DoxyItemize}
\item Schwarzschild\+: static observer.
\item Kerr\+: ZAMO by default (static only where timelike, i.\+e., g\+\_\+tt \texorpdfstring{$<$}{<} 0). Document the choice in metadata.
\end{DoxyItemize}
\end{DoxyItemize}

\hypertarget{md__p_l_a_n_autotoc_md25}{}\doxysection{\texorpdfstring{2) Accuracy Targets (Research Mode)}{2) Accuracy Targets (Research Mode)}}\label{md__p_l_a_n_autotoc_md25}
Schwarzschild
\begin{DoxyItemize}
\item r\+\_\+min(b)\+: relative error ≤ 1e−12 for b ≥ 1.\+05 b\+\_\+crit; ≤ 1e−6 down to 1.\+01 b\+\_\+crit.
\item b\+\_\+crit\+: \texorpdfstring{$\vert$}{|}b\+\_\+num − 3√3 M\texorpdfstring{$\vert$}{|} / (3√3 M) ≤ 1e−10.
\item Deflection α(b)\+:
\begin{DoxyItemize}
\item Weak field (b ≥ 20 M)\+: \texorpdfstring{$\vert$}{|}α\+\_\+num − α\+\_\+\+PN\texorpdfstring{$\vert$}{|} ≤ 1e−3.
\item Near-\/critical fit α(b) ≈ −c1 ln(b/b\+\_\+crit − 1) + c2\+: c1 within 1\% of the known analytic value; c2 within 2\%.
\end{DoxyItemize}
\item Invariants drift along accepted steps\+: \texorpdfstring{$\vert$}{|}\+H\texorpdfstring{$\vert$}{|}max ≤ 1e−12, \texorpdfstring{$\vert$}{|}Δ\+E/\+E\texorpdfstring{$\vert$}{|}, \texorpdfstring{$\vert$}{|}Δ\+L/\+E\texorpdfstring{$\vert$}{|} ≤ 1e−12.
\end{DoxyItemize}

Kerr
\begin{DoxyItemize}
\item Match reference tool (Geo\+Kerr) over a modest grid of (a/M, b/M, θ\+\_\+obs)\+:
\begin{DoxyItemize}
\item Δφ agreement ≤ 1e−6 rad (equatorial); ≤ 1e−5 off‑equatorial.
\item Identical fate (escape/capture/spherical) classification.
\end{DoxyItemize}
\item Invariants with Carter constant Q\+: \texorpdfstring{$\vert$}{|}\+H\texorpdfstring{$\vert$}{|}max ≤ 1e−12, \texorpdfstring{$\vert$}{|}Δ\+E/\+E\texorpdfstring{$\vert$}{|}, \texorpdfstring{$\vert$}{|}Δ\+Lz/\+E\texorpdfstring{$\vert$}{|}, \texorpdfstring{$\vert$}{|}Δ\+Q/\+Q\texorpdfstring{$\vert$}{|} ≤ 1e−10.
\end{DoxyItemize}

Renderer (when relevant)
\begin{DoxyItemize}
\item Equatorial thin disk intersections reproducible to ≤ 1e−9 r.
\item Golden images PSNR ≥ 60 dB relative to research baseline (identical seeds/settings).
\end{DoxyItemize}

\hypertarget{md__p_l_a_n_autotoc_md27}{}\doxysection{\texorpdfstring{3) Architecture Choices (Research Mode)}{3) Architecture Choices (Research Mode)}}\label{md__p_l_a_n_autotoc_md27}

\begin{DoxyItemize}
\item Precision\+: FP64 in all geodesic math and event detection; FP32 allowed only for shading and final buffer packing.
\item Integrator
\begin{DoxyItemize}
\item Schwarzschild\+: Hamiltonian RK with analytic turning radius and potential‑aware events; optional exact quadrature for φ(r) to audit α(b).
\item Kerr\+: Prefer separated form in Mino time (λ̃) using Carter potentials R(r), Θ(θ) for research mode; keep Hamiltonian form available for cross‑checks.
\end{DoxyItemize}
\item Coordinates
\begin{DoxyItemize}
\item Use BL for far field; near the horizon or when g\+\_\+tt → 0 (Kerr), prefer horizon‑regular forms (Eddington–\+Finkelstein or Kerr–\+Schild) or terminate with clear classification before ill‑conditioning.
\end{DoxyItemize}
\item Adaptive error control
\begin{DoxyItemize}
\item Embedded high‑order method (RKF78 / DOP853) or step‑doubling with strict rejection; invariant‑gated acceptance (H, E, Lz, Q).
\end{DoxyItemize}
\item Events and classification
\begin{DoxyItemize}
\item Escape\+: asymptotic check (monotone r beyond r\+\_\+esc and positive radial potential), not just r threshold.
\item Turning points\+: explicit detection from potentials; analytic turning radius in Schwarzschild, robust root‑finding in Kerr.
\item Disk intersections\+: root‑find θ − π/2 between brackets with interpolation safe-\/guards; sort multi‑hits; self‑occlusion optional.
\end{DoxyItemize}
\end{DoxyItemize}

\hypertarget{md__p_l_a_n_autotoc_md29}{}\doxysection{\texorpdfstring{4) Workstreams and Deliverables}{4) Workstreams and Deliverables}}\label{md__p_l_a_n_autotoc_md29}
WS‑0 — Research Mode Baseline and Governance (1–2 days)
\begin{DoxyItemize}
\item Add a single “research” mode switch that forces FP64, disables fast‑math, and sets strict integrator tolerances and invariant gates.
\item CLI\+: {\ttfamily -\/-\/mode research} is the only documented mode in this plan.
\item Metadata JSON sidecar for every product\+: commit, compile flags, GPU, CC, driver, integrator kind/order, tolerances, scene params, RNG seed, timing.
\item Deliverable\+: example run produces image/table + JSON; deterministic across runs on the same GPU/driver.
\end{DoxyItemize}

WS‑1 — Schwarzschild Accuracy Pack (4–6 days)
\begin{DoxyEnumerate}
\item Exact turning radius and classification
\begin{DoxyItemize}
\item Implement closed‑form r\+\_\+min(b) from cubic r\texorpdfstring{$^\wedge$}{\string^}3 − b\texorpdfstring{$^\wedge$}{\string^}2 r + b\texorpdfstring{$^\wedge$}{\string^}2 Rs = 0 for b \texorpdfstring{$>$}{>} b\+\_\+crit.
\item Output arrays\+: turning\+\_\+radius (NaN if none), closest\+\_\+radius\+\_\+reached (for captured/diagnostics).
\end{DoxyItemize}
\item Escape and deflection
\begin{DoxyItemize}
\item Replace “r\texorpdfstring{$>$}{>}threshold” with potential‑aware escape\+: require r≥r\+\_\+esc and monotone increase with R(r)\texorpdfstring{$>$}{>}0 for k consecutive steps.
\item Validate α(b) with PN series at large b and strong‑deflection fit near b\+\_\+crit (Bozza)\+: report residuals.
\end{DoxyItemize}
\item Initial condition independence
\begin{DoxyItemize}
\item Demonstrate α(b) invariance for r0∈\{500,1000,2000\} Rs (Δα ≤ 1e−6). Prefer turning‑point init for audit path.
\end{DoxyItemize}
\item Convergence \& invariants
\begin{DoxyItemize}
\item Adaptive stepping + invariant gates (H, E, L) with reject/halve on violation; plot error vs. step.
\end{DoxyItemize}
\item Tests \& artifacts
\begin{DoxyItemize}
\item Unit tests\+: b\+\_\+crit, r\+\_\+min(b) grid, α(b) vs PN/strong‑deflection; convergence slopes.
\item Artifacts\+: CSV tables, plots (saved via scripts), and a “\+Schwarzschild validation” README with figures.
\end{DoxyItemize}
\end{DoxyEnumerate}

WS‑2 — Kerr Accuracy Pack (8–12 days)
\begin{DoxyEnumerate}
\item Separated equations in Mino time
\begin{DoxyItemize}
\item Implement R(r), Θ(θ) potentials, Carter constant Q; integrate dφ/dλ̃, dt/dλ̃ alongside dr/dλ̃, dθ/dλ̃.
\item Robust root‑finding at turning points; event functions for spherical photon orbits (R=0, d\+R/dr=0).
\end{DoxyItemize}
\item Observer and emitter frames
\begin{DoxyItemize}
\item Use ZAMO tetrad by default. Provide static only where g\+\_\+tt \texorpdfstring{$<$}{<} 0 (assert otherwise).
\item Disk inner radius r\+\_\+in = r\+\_\+isco(a) (analytic expression); expose override.
\end{DoxyItemize}
\item Validation grid
\begin{DoxyItemize}
\item Build a modest set of cases (e.\+g., a/M ∈ \{0,0.\+5,0.\+9,0.\+99\}, b/M spanning near‑critical to weak‑field; θ\+\_\+obs ∈ \{π/2, π/3\}).
\item Cross‑validate Δφ, turning radii, and fates vs Geo\+Kerr.
\end{DoxyItemize}
\item Invariants and convergence
\begin{DoxyItemize}
\item Track H, E, Lz, Q drift; require research‑mode thresholds. Show p‑order slopes.
\end{DoxyItemize}
\item Tests \& artifacts
\begin{DoxyItemize}
\item GPU tests gated by env; reference CSV from Geo\+Kerr runs (stored under {\ttfamily geokerr\+\_\+port/validation/test\+\_\+data/}).
\item Plots and summary tables committed.
\end{DoxyItemize}
\end{DoxyEnumerate}

WS‑3 — Research‑\+Grade Disk Rendering (Optional, 4–6 days)
\begin{DoxyEnumerate}
\item Disk geometry and intersections
\begin{DoxyItemize}
\item Multi‑hit traversal; sort by affine parameter; self‑occlusion enabled.
\item Robust equatorial crossing root‑find with angle guarding.
\end{DoxyItemize}
\item Radiometry
\begin{DoxyItemize}
\item I\+\_\+obs = g\texorpdfstring{$^\wedge$}{\string^}3 I\+\_\+em with I\+\_\+em(r) = r\texorpdfstring{$^\wedge$}{\string^}−p; record p in metadata. Optionally add banded outputs (grayscale per ν band).
\end{DoxyItemize}
\item Golden images
\begin{DoxyItemize}
\item Fixed seeds, metadata, PSNR ≥ 60 dB across reruns. Provide reference PNG/\+EXR assets.
\end{DoxyItemize}
\end{DoxyEnumerate}

WS‑4 — Reproducibility \& Tooling (2–3 days)

\item Experiment manifests (YAML/\+JSON)\+: end‑to‑end reproducible configurations to regenerate figures.

\hypertarget{md__p_l_a_n_autotoc_md33}{}\doxysection{\texorpdfstring{6) Numerical \& GPU Considerations}{6) Numerical \& GPU Considerations}}\label{md__p_l_a_n_autotoc_md33}
Precision and math
\begin{DoxyItemize}
\item FP64 kernels for geodesics; require precise division/sqrt; prefer fused FMA.
\item Avoid FTZ/\+DAZ for FP64 segments. Clamp only where mathematically justified (e.\+g., sin θ → sign‑preserving small value).
\end{DoxyItemize}

Step control
\begin{DoxyItemize}
\item Research mode\+: mandatory adaptive with strict tolerances (e.\+g., rel 1e−12, abs 1e−14) and invariant gates.
\item Quantize accepted h only if it does not inflate error; otherwise leave continuous.
\end{DoxyItemize}

Event robustness
\begin{DoxyItemize}
\item Use bracketed root‑finders (bisection + secant) for events (turning points, θ=π/2 crossing) to guarantee convergence.
\end{DoxyItemize}

Parallelism and determinism
\begin{DoxyItemize}
\item One ray per thread; avoid race conditions; diagnostical atomics are okay but not used to alter physics decisions.
\item Deterministic sampling\+: fixed seeds from pixel coords and pass index.
\end{DoxyItemize}

Memory and performance
\begin{DoxyItemize}
\item Keep pinned host buffers and tiling; overlap compute and transfers. Performance is secondary to accuracy but we should avoid pathological slowdowns.
\end{DoxyItemize}
